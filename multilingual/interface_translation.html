If you want to translate a node by hand simply type your translation of the source to the corresponding field. If the language is similar ie. Fernch to canadian French, you can first copy the the source to the field then edit.

If you want to use a machine translation browser plugin simply install the plugin (available from chrome web store, mozilla.com, IOS app store etc.). Once you have the plugin, translate the page to the desired language. You will notice that the only thing that changes on the page are the "copy" buttons above the source text. Click the copy button to put the translated text in the field, then continue editing.

Note: trasnslation best practice is to always have machine-translated text carefully reviewed and edited by a human translator.

Searches may be limited to strings in a specific language.
